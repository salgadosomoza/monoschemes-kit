Usage

As <img> tag

<img src="src/icons/search.svg"
     width="24" height="24"
     alt="search">

Inline SVG (color control)

<svg width="24" height="24"
     viewBox="0 0 24 24"
     fill="currentColor"
     style="color: #2B2F36">
  <path d="..."/>
</svg>

CSS size classes

.icon-sm  { width: 16px; height: 16px; }
.icon     { width: 24px; height: 24px; }
.icon-md  { width: 32px; height: 32px; }
.icon-lg  { width: 48px; height: 48px; }
.icon-xl  { width: 64px; height: 64px; }
